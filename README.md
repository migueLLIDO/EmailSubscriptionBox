# Box de Inscrição por E-mail

Componente de newsletter com ícone, texto de convite, campo de e-mail e botão de subscribe.

Este projeto é uma demonstração estática feita com **HTML e CSS**, pensada para funcionar diretamente no navegador e poder ser publicada com **GitHub Pages**.

## Funcionalidades

- formulário simples
- validação nativa de e-mail
- estilo visual em camadas
- mensagem anti-spam

## Estrutura do projeto

- `css/style.css`
- `img/icone.png`
- `index.html`

## Visual

O projeto usa um box branco com topo azul e detalhes roxo/azul.

## Como executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente no navegador.
3. Se preferir usar um servidor local, execute na pasta do projeto:

```bash
python -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```

## GitHub Pages

Como este projeto é estático, ele pode ser publicado no GitHub Pages usando a branch principal.

Configuração recomendada:

- Branch: `main`
- Pasta: `/ (root)`

## Tecnologias

- HTML5
- CSS3
- Font Awesome
